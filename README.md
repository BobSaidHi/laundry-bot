# laundry-bot
This bot will find the optimal time to do one's laundry at the California Polytechnic State University, San Luis Obispo.

## Update

The old software with a scrape-able, public facing webpage was retired a year or two ago. It has been replaced with Campus Clean Pay and/or the WASH app, which are mobile apps and require frequent authentication with an account.

Since I haven't worked on this for a while, I am archiving this repository for now.
